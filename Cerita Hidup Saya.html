Cerita Hidup Saya

    Tangal Lahir : 07 Januari 1996

    Alamat : Banjarmasin

    Cerita Saya

    Saya berasal dari keluarga sederhana yang berkecukupan dan saya anak kedua dari dua bersaudara disini saya akan bercerita sedikit tentang hidup saya, disaat umur saya 7 tahun saya dijumpai dengan makhluk akstral yang tidak bisa dilihat oleh orang yang biasa disitu saya mau diangkat sebagai anak nya disitu saya menolak dengan tegas disaat itu dia pun marah dan tiba-tiba menghilang setelah beberapa tahun kemudian ada perubahan dihidup saya disitu saya dihantui dengan rasa takut terus menerus setelah umur saya sudah cukup tua saya bisa melihat makhluk-makhluk akstral yang menjumpai saya ataupun dimana saya berada dengan semua itu saya pun becerita dengan keluarga terdekat saya dan mereka tidak mempercayi dengan apa yang terjadi kepada saya lama kelamaan keluarga saya bisa menerima apa yang saya rasakan selama ini, disemua itu saya merasa tertekan oleh karena dihampiri dengan makhluk akstral terus menerus sampai saat ini pun saya bisa melihat hal-hal mistis seperti itu. Itulah cerita saya yang saya alami selama ini dan saat ini.
